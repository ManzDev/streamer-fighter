# Phaser

## Scenes

```js
this.scene.scene.launch();    // ejecuta en paralelo
this.scene.scene.run();       // ejecuta pero no cambia estado
this.scene.scene.resume();    // arranca update de la escena
this.scene.scene.setActive(); //
this.scene.scene.start();     // ejecuta y para la actual
this.scene.scene.switch();    // ejecuta y duerme la actual
```
